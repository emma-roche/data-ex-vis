# 🎨 Visualization & Aesthetics

### Image Exploration

* Computer Vision Basics: Understanding RGB channels, pixel intensity, and histograms.
* Feature Extraction: Identifying edges, textures, or shapes using filters.
* Dimensionality Reduction: Using PCA or t-SNE to visualize high-dimensional image clusters.
* Metadata Extraction: Accessing EXIF data and timestamps.

### Interactive Visualization (Theory & Plotly)
* Grammar of Graphics: The logic behind mapping data to visual aesthetics.
* Human Perception: Gestalt principles and color theory (avoiding "rainbow" maps).
* Declarative vs. Imperative: Understanding the Plotly Express API.
* Callbacks: Creating "linked brushing" where hovering over one plot updates another.

## Virtual Environments

There is an already installed uv environment for this topic.
In order to use it in Jupyter Notebook/Lab, you need to add it as a kernel. To do this, run the following command:
```# activate the environment
source /v/courses/2026-dataexplorationandvisualization2026.public/uv_envs/<env_name>/bin/activate
# install the kernel
python -m ipykernel install --user --name=interactive --display-name "Python (interactive)"
```
After running the above command, you should see the new kernel option "Python (interactive)" when you create a new notebook in Jupyter Notebook/Lab. You can select this kernel to use the virtual environment for your image exploration tasks.
You might have to reload the page first to see the new kernel option.
