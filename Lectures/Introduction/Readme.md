## Course overview
This course is designed to provide students with a comprehensive understanding of data exploration 
* Retrieving data from various sources (such as databases, APIs, and web scraping)
* Working with various data types (like numerical, image, and text data)
* Working with different data formats (like CSV, JSON, and Excel)
* Handling missing data and outliers
* Cleaning and preprocessing data
* Analyzing data using statistical methods

and visualization techniques
* Creating effective visualizations to communicate insights
* Using tools like Matplotlib, Seaborn, and Plotly for data visualization

Also covering best practices for making sure that the work done is reproducible and shareable, including:
* Using version control systems like Git
* Documenting code and analysis
* Sharing notebooks and visualizations with others
* Using virtual environments to manage dependencies with tools like `uv` (`conda` or `venv` can also be used)

There will be a emphasis on AI assisted workflows, where students will learn how to leverage AI tools to enhance their data exploration and visualization skills.
* Github Copilot for code generation and suggestions
* AI agents for exploring data sources via MCP (Model Context Protocol) servers

# Datascience 
When do we generate data?
* In experiments (raw data, analysed data)
* In simulations (output of softwares)
* In logs of software (usage, errors, performance)
* In social media (posts, interactions)

What do we do with data?
* Store it (database, files)
* Analyse it (clean, filter, transform, reduce)
* Share it to collaborators (structure, logic)
* Support published results with data (provenance)
